# Changelog

## 13.0.0

- Initial release: Markdown Importer for Foundry VTT v13.0.0
- Injects a "Convert Markdown" button directly into the ProseMirror menu for manual conversion.
- Adds a client setting to automatically convert Markdown syntax to rich text on each journal page save (disabled by default).
- System-agnostic module compatible with any Foundry VTT game system.
